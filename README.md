# 🍕 Pizza Dome

**Street-style artisan pizza with a premium twist.**

Pizza Dome is a modern, high-performance web application for an artisan pizza brand. It features a stunning, responsive UI with dark mode support, a live kitchen visualizer, and a seamless checkout experience.

## ✨ Features

- 🌓 **Dynamic Dark Mode**: Beautifully crafted themes for day and night.
- 🎥 **Live Kitchen Feed**: Integrated video feed of the "Open Kitchen".
- 🛒 **Interactive Menu**: Fully customizable pizzas with size and add-on selections.
- 📱 **Mobile Optimized**: Perfectly responsive design for all devices.
- 💳 **Secure Checkout**: Built-in payment modal with UPI, Card, and Wallet support.
- ⚡ **Vite Powered**: Blazing fast performance and developer experience.

## 🚀 Tech Stack

- **Frontend**: React.js, Vite
- **Styling**: Vanilla CSS with modern Flexbox/Grid
- **Fonts**: Syne & DM Sans (via Google Fonts)
- **Icons**: Fluent Emojis

## 🛠️ Setup & Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ARJUNARROW/PizzaDome.git
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

## 📁 Project Structure

- `PizzaDome.jsx`: Main application component containing all UI logic.
- `public/`: Static assets including videos and images.
- `src/`: Source files and entry points.
