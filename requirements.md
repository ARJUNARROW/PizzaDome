# 📋 Project Requirements

This document outlines the software and configuration requirements to run the Pizza Dome project.

## 💻 Software Requirements

- **Node.js**: v16.x or higher (Recommended: v20.x LTS)
- **npm**: v8.x or higher
- **MongoDB**: v6.0 or higher (Running locally or on Atlas)
- **Web Browser**: Chrome, Firefox, or Safari (Modern versions)

## 🔑 API Keys & Accounts

- **Razorpay Account**: Required for payment processing.
  - `RAZORPAY_KEY_ID`: Your test/live key ID.
  - `RAZORPAY_KEY_SECRET`: Your test/live secret key.

## 📦 Key Dependencies

### Frontend
- **React**: UI library
- **Vite**: Build tool & dev server
- **Syne & DM Sans**: UI Typography

### Backend
- **Express**: Web framework
- **Mongoose**: MongoDB object modeling
- **Razorpay SDK**: Payment integration
- **CORS**: Cross-origin resource sharing
- **Dotenv**: Environment variable management
